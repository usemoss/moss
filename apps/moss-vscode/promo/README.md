# Moss Code Search — launch promo

~50-second music-driven promo for **Moss Code Search** — the VS Code / Cursor extension powered by [Moss](https://moss.dev).

## Story

1. Hero — "You know the code. Just not where it lives."
2. Grep failure — Find in Files returns the wrong file (inside a pixel-accurate VS Code window)
3. **How it works** — Scan → Chunk → Embed on-device → Query in ~3ms
4. Product demo — Create Index → semantic query → click result → jump to code
5. **In your editor** — Live search, click to jump, stays indexed, cloud optional
6. Outro — VS Code + Moss logos crash → **VS Code Finder** → Available on moss.dev → Powered by Moss

## Commands

```bash
npm install
npm run dev      # Remotion Studio
npm run render   # → out/moss-vscode-promo.mp4
```

## Specs

- 1920×1080, 30fps, ~50 seconds (1496 frames)
- Music + SFX only (no voiceover)
- Self-contained under `apps/moss-vscode/promo/`
