Moss Browser SDK Examples

Examples for @moss-dev/moss-web - in-browser semantic search powered by WASM.

  • Load & Query - Load an existing index and run searches
  • Comprehensive - Full CRUD lifecycle: create, add, query, delete
  • Metadata Filtering - Query with $eq, $and, $or, $lt, $in, and $near filters

Setup

  1. Run npm install
  2. Run npm run dev
  3. Open this page in your browser and pick an example
